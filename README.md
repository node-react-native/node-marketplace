# node-marketplace
REST API backend with NodeJS

# pre-requisites

1 - Install Docker Desktop
2 - Install Node
3 - Install Yarn

# run

1 - Clone repository
2 - Run: docker run --name myinstance -p 27017:27017 -d -t mongo
3 - Run: docker run --name noderedis -p 6379:6379 -d -t redis:alpine
2 - Run: yarn start
